## ğŸš€ **Quick Start**

### **Local Development**

1. **Clone the repository:**
   ```bash
   git clone https://github.com/DevShubham07/PeoplePulse.git
   cd PeoplePulse
   ```

2. **Start Backend:**
   ```bash
   cd backend
   ./mvnw spring-boot:run
   ```

3. **Start Frontend:**
   ```bash
   cd frontend
   npm install
   npm start
   ```

4. **Access the application:**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:8080
